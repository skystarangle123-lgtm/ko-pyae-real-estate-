# Ko Pyae Real Estate – AI-Powered CRM

🏛️ **Premium Real Estate Management System** for Myanmar's real estate market.

## ✨ Features

### 🤖 AI-Powered Classification
- Automatic data extraction and categorization
- Myanmar & English language support
- Phone number normalization
- Smart field extraction

### 📱 Progressive Web App (PWA)
- Offline-first architecture
- Service worker caching
- Installable on mobile & desktop
- Sync when connection returns

### 💾 Data Management
- IndexedDB local storage
- CSV/JSON export
- Full record management
- Search & filter capabilities

### 🎨 Beautiful UI
- Luxury real estate branding
- Royal blue + gold theme
- Responsive design
- Smooth animations

## 🚀 Quick Start

### Local Development

```bash
# Clone repository
git clone https://github.com/skystarangle123-lgtm/ko-pyae-real-estate-.git
cd ko-pyae-real-estate-

# Serve locally (choose one)
python -m http.server 8000
# OR
npx http-server

# Open browser
open http://localhost:8000
```

### Deploy to Vercel

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel
```

## 💡 Usage

1. **Enter Data** - Type property listings, client info, or appointments in Myanmar or English
2. **AI Classifies** - System automatically extracts information and categorizes
3. **Review & Save** - Confirm the extracted data and save to database
4. **Export** - Export records as CSV or JSON

## 🛠️ Technology Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Storage**: IndexedDB (via window.storage API)
- **PWA**: Service Workers, Web Manifest
- **Hosting**: Vercel
- **Language**: Myanmar (Burmese) & English

## 📊 Data Categories

- **Rentals** 🏠 - Apartment/house rental listings
- **Sales** 🏢 - Property for sale listings
- **Clients** 👤 - Buyer/tenant inquiries
- **Schedules** 📅 - Meetings and viewings
